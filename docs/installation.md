# Installation

You can clone our repository using Git:

```bash
git clone https://github.com/light156/multi-ancestry-COJO.git
cd multi-ancestry-COJO
```

Or download our software program directly from the links below:

### [⬇️ Download link (Linux)](https://github.com/light156/multi-ancestry-COJO/releases/download/v1.0.0/manc_cojo)

### [⬇️ Download link (Windows)](https://github.com/light156/multi-ancestry-COJO/releases/download/v1.0.0/manc_cojo_win.exe)

### [⬇️ Download link (macOS)](https://github.com/light156/multi-ancestry-COJO/releases/download/v1.0.0/manc_cojo_macOS)

<br> 

On **Linux** and **macOS**, you may need to give the file permission to run:

```bash
chmod +x manc_cojo
```

To check that the program is working, run:

```bash
./manc_cojo --help
```

This will print the available options and confirm that the program runs correctly.

---

If you have any problems running the software on your system, you may
- Try building from source code by following the steps in [Building from Source](https://light156.github.io/multi-ancestry-COJO-docs/build/) (which is quite simple).
- Feel free to reach out and we are very happy to help.
